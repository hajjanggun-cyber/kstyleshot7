# Post Image Prompt Pack (Tier1 Calendar 20)

Updated At (KST): 2026-03-05 19:02:00 +09:00

## Usage Rule

- One hero image per post.
- Recommended ratio: `16:9` (1200x675 or 1600x900).
- Keep visual style consistent across the 4-week run.
- Avoid logos, brand marks, readable text overlays, and celebrity likeness.

## Base Style Prefix

Use this prefix at the start of every prompt:

`Cinematic Korean editorial portrait photography, clean skin texture, natural light, realistic color grading, premium fashion magazine look, high detail, no text, no watermark, no logo`

## Negative Prompt (Shared)

`low quality, blurry face, extra fingers, duplicated face, distorted anatomy, watermark, text overlay, logo, overprocessed skin, oversaturated neon, noisy background`
