# codex-pildok.md — Codex 작업 전 필독 파일 목록

어떤 작업을 시작하든 아래 3개 파일을 **반드시 먼저 읽는다.**
읽지 않고 작업을 시작하지 않는다.

---

## 필독 순서

### 1. `md-doc/google-seo-guide.md`
Google 공식 SEO 기준. 글 작성 품질의 최상위 기준이다.
- E-E-A-T 원칙
- title / description 길이 기준 (60자 / 120~155자)
- AI 생성 콘텐츠 정책
- 검색엔진 중심 콘텐츠 금지 항목

### 2. `AGENTS.md`
출력 스펙 · 품질 체크리스트 · 법적 제한 사항.
- frontmatter 필드 정의
- H2 em dash 형식 (하이픈 금지)
- 이미지 플레이스홀더 규칙
- 내부 링크 규칙 (언어 격리 포함)
- 법적 금지 항목 (아이돌명, 기획사명, 곡명 등)
- Quality Gate 체크리스트

### 3. `md-doc/post-codex-command.md`
작업 순서 · 진행 로그 · 프롬프트 모음.
- [작업 진행 로그] — 어디까지 완료됐는지 확인
- [전체 작성 순서] — 다음 작성할 파일 결정
- [A] 일반 세션 / [B] 허브 완료 후 작업 프롬프트
