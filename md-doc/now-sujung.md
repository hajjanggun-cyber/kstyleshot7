# Now Sujung

기존에 작성한 글을 보고 아래 패턴대로 수정한다.

- KO: `https://kstyleshot.com/ko/hub/[slug]`
- EN: `https://kstyleshot.com/en/hub/[slug]`

한 줄 결론:

그 GPT 의견을 100% 따르기보다, 키워드 구조화, 제목 개선, 이미지 문맥 강화만 취하고 과한 SEO 장식은 버리는 게 맞다.

실무적으로는 이 정도가 최선이다.

- 제목은 더 검색 친화적으로 다듬기
- 첫 문단에 핵심 키워드 명확히 넣기
- H2를 조금 더 검색형으로 조정하기
- 포토존 글은 리스트형 구조 강화
- 이미지 alt와 주변 문맥 보강
- 기존 글은 최대한 살리면서 문맥 가져가기

## 키워드 원칙

- 메인 키워드: 이 글이 대표로 가져갈 검색어 1개
- 하위 키워드: 메인 키워드를 보조하면서 검색 의도를 넓혀주는 표현 3~5개
- 금지: 같은 허브 안에서 다른 글의 메인 키워드를 뺏어오는 것

예를 들어 경복궁 포토 가이드라면:

- 메인 키워드
  - KO: `경복궁 포토존 추천`
  - EN: `Best photo spots in Gyeongbokgung Palace`
- 하위 키워드
  - KO: `경복궁 사진 명소`, `경복궁 한복 사진`, `경복궁 사진 구도`, `경복궁 인생샷`, `경복궁 촬영 포인트`
  - EN: `Gyeongbokgung Palace photography guide`, `hanbok photos in Gyeongbokgung`, `best angles in Gyeongbokgung Palace`, `Seoul palace photo spots`, `where to take photos in Gyeongbokgung`

## 키워드 배치 원칙

이걸 어디에 넣느냐가 중요하다.

- 제목: 메인 키워드
- 첫 문단: 메인 키워드 + 하위 키워드 1~2개
- H2 2개 정도: 하위 키워드 반영
- 이미지 alt/주변 문단: 장소형 하위 키워드 반영
- 마무리/요약: 메인 키워드 1회 자연스럽게 회수

즉, 단순히 키워드를 더 넣자가 아니라, 이 글이 어떤 검색어를 대표할지 명확히 하고 나머지 관련 검색어를 주변부로 받쳐주는 구조가 필요하다.
